# Bank RD Account Management System

## Overview
This is a Java console application that simulates a Recurring Deposit (RD) Account Management System. The program validates user input, calculates the total deposited amount, interest earned, and the final maturity amount based on the customer's age, gender, and tenure.

## Features
- Accepts monthly deposit amount
- Validates user inputs
- Calculates total deposited amount
- Calculates interest based on age and gender
- Calculates maturity amount
- Uses custom exception handling
- Demonstrates abstraction and inheritance concepts in Java

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
